Example: "Smart Password Categories" Feature
The Problem
Right now, when users save passwords, you probably have them select a category manually (like "Social Media," "Banking," "Email," etc.) or type it in.

The AI-Powered Solution with Genkit
Genkit can automatically detect and categorize passwords when users save them, saving them time and keeping things organized.

Without Genkit (The Hard Way)
You'd need to:

Learn a specific AI API (like OpenAI's GPT)

Write complex prompt engineering code

Manually parse JSON responses (which might be malformed)

Handle errors when the AI returns weird formats

Build a testing system to try different prompts

Deploy and monitor with custom logging

javascript
// Without Genkit - messy, fragile, provider-specific
async function categorizeWebsite(url) {
    const openai = new OpenAI({ apiKey: process.env.OPENAI_KEY });
    
    try {
        const response = await openai.chat.completions.create({
            model: "gpt-3.5-turbo",
            messages: [{
                role: "user",
                content: `Categorize this website: ${url}. 
                         Return ONLY one word: banking, social, email, shopping, or other.`
            }]
        });
        
        // Parse the response - but what if it's formatted weird?
        const category = response.choices[0].message.content.trim().toLowerCase();
        
        // Validate it's one of your expected categories
        const validCategories = ['banking', 'social', 'email', 'shopping', 'other'];
        return validCategories.includes(category) ? category : 'other';
        
    } catch (error) {
        console.error('AI failed:', error);
        return 'other'; // Fallback is manual entry
    }
}
With Genkit (The Smart Way)
javascript
import { genkit } from 'genkit';
import { googleAI, gemini15Flash } from '@genkit-ai/googleai';
import { z } from 'zod';

// Set up Genkit once
const ai = genkit({
    plugins: [googleAI()],
});

// Define exactly what you want back
const CategorySchema = z.object({
    category: z.enum(['banking', 'social', 'email', 'shopping', 'other']),
    confidence: z.number().min(0).max(100),
    suggestedName: z.string().optional()
});

// Your clean, maintainable function
async function smartCategorize(url, siteName) {
    const { output } = await ai.generate({
        model: gemini15Flash,
        prompt: `Analyze this website and categorize it appropriately:
                 URL: ${url}
                 Site Name: ${siteName}`,
        output: { schema: CategorySchema }, // Genkit ensures valid format!
    });
    
    return output; // Always has the exact shape you defined
}

// In your password save screen
const handleSavePassword = async (url, username, password) => {
    // Get AI suggestions automatically
    const suggestion = await smartCategorize(url, window.location.hostname);
    
    // Pre-fill the form with AI suggestions
    setCategory(suggestion.category);
    showConfidenceBadge(suggestion.confidence); // "90% confident this is banking"
    
    // User can override if AI is wrong
};
